# KNIME Workflows

Rapid prototyping and stakeholder communication workflows.

## casino_etl_prototype.knwf
Visual prototype of the casino data pipeline before production implementation.

**Use case:** Demonstrated data flow to business stakeholders, validated logic, 
then implemented in PySpark for production scale.

**Benefits:**
- Shortened feedback loop from weeks to days
- Non-technical stakeholders could review pipeline logic
- Caught edge cases before writing production code
